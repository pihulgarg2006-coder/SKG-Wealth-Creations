---

## 💡 JavaScript Features (in index.js)

- **Typing Animation** — rotates through taglines character by character (based on Module 7 reference)
- **Blinking Cursor** — toggles visibility with `setInterval`
- **Navbar Scroll Effect** — adds background on scroll
- **Hamburger Menu** — mobile responsive nav toggle
- **Scroll Fade-In** — `IntersectionObserver` to reveal cards on scroll
- **Stats Counter** — animated number count-up when section enters viewport
- **Form Validation** — validates name, phone, and service before submission

## 🚀 How to Run Locally

1. Download or clone this repo
2. Open the folder in **Visual Studio Code**
3. Install the **Live Server** extension
4. Right-click `index.html` → **Open with Live Server**
